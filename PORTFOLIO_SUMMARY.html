🚀 Quick Start Guide

1. Test Locally

Open index.html in any modern browser Test on different screen sizes Check animations and interactions Use browser DevTools (F12) to inspect

2. Customize Content

Edit text content in HTML Update project links Change social media links Update contact information Add your actual GitHub/LinkedIn URLs

3. Customize Colors

Change --accent-primary in style.css Update --accent-secondary for secondary color Modify background colors if desired All colors in :root for easy access

4. Deploy to GitHub Pages

1. Push files to GitHub repository 2. Go to Settings > Pages 3. Select main branch as source 4. Enable custom domain if available 5. Site will be live in minutes
